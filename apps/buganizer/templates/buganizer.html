External Security Report: Reflected Cross-Site Scripting (XSS) in Legacy ERP Portal

SD

reply more_vert

Executive Summary


An external security researcher has reported a Reflected Cross-Site Scripting (XSS) vulnerability within the Legacy Enterprise Portal. Initial assessment confirms that user-supplied input is reflected in the HTTP response without adequate sanitization, potentially allowing arbitrary code execution in the victim's browser.

Affected Asset


  • Asset: Legacy Enterprise Portal (Local deployment)
  • Vulnerability Type: Reflected XSS

Description


The application accepts user-controlled input and reflects it back to the client. If an attacker crafts a link with malicious JavaScript, it will execute in the context of the user's session when visited.

Impact


If successfully exploited, this vulnerability could allow an attacker to access the sensitive session cookie (auth_token) using document.cookie, enabling them to hijack the administrator session.
